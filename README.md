# Задание 1

## Задача 1

### Текст задачи

Дерево содержит вещественные числа. Заменить отрицательные значения на
0, положительные — на 1.

### Алгоритм решения

Задали дерево со значениями, вывели для показа, результат получаем через функцию smena, она сравнивает дерево если его нету то его нету, если есть она берет значение и заменяет его, если меньше нуля то нуль, если больше то 1, если нет то оставляет такое, и перевызывается.

создали дерево, показали его, вводим цифру для поиска, проверка на ввод, создаем функцию подсчета, которая перебирает узлы дерева, и если находит нужное число перебирает цифры и при нахождении добавляет 1 к счетчику
